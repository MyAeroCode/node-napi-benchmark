### How to build it?

**pre-build**

1. `npm install -g windows-build-tools` in admin mode.

**build**

1. `npm install`
2. `npm run build`
3. `npm run start`

---

### Http Get Request Parallel

Connect N connections simultaneously to a given site.

**Library :**

c++ :

1. `libcurl`

node :

1. `sync-request`
2. `axios`
